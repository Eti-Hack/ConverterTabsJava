# ConverterTabsJava 📱

Application Android de conversion d'unités avec navigation par onglets.
Développée en Java avec Android Studio.

---

## 📋 Description

ConverterTabsJava est une application mobile Android qui permet de convertir :
- 🌡️ **Températures** : Celsius ↔ Fahrenheit
- 📏 **Distances** : Kilomètres ↔ Miles

La navigation entre les deux convertisseurs se fait via des **onglets** (TabLayout + ViewPager2).

---

## 🧱 Technologies utilisées

| Technologie | Version | Rôle |
|-------------|---------|------|
| Java | 11 | Langage principal |
| Android SDK | 34 | Plateforme cible |
| Material Design | 1.13.0 | Composants UI (TabLayout) |
| ViewPager2 | 1.1.0 | Navigation entre fragments |
| AndroidX AppCompat | — | Compatibilité arrière |
